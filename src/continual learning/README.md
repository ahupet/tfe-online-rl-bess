# Continual Learning & Long-Term Evaluation Submodule

This folder encompasses the scripts used to evaluate agent robustness across extended multi-year horizons (2020–2023) and contains the online fine-tuning frameworks designed to mitigate catastrophic forgetting.

## 📂 Directory Contents

### Long-Term Baselines (Frozen Weights)
* **`TD3_long_term_evaluation.py`**: Performance assessment script for the static `TD3` model, compiling annual Latex tables and exporting sorted multi-sheet Excel files (Daily, Monthly, and Annual returns).
* **`PPO_long_term_evaluation.py`**: Performance assessment script for the static `PPO` model, handling hidden recurrent stochastic status parameters across multi-year data.

### Online Fine-Tuning Loops
* **`PPO_continual_learning_without_stairs.py`**: Standard time-triggered online adaptation pipeline. It evaluates chronological blocks (Weekly/Monthly), fine-tunes weights based on experienced transitions, and rolls forward.
* **`PPO_Continual_ADAPT.py`**: The adaptive gating framework. It dynamically evaluates previous market variance; if volatility drops below a defensive threshold of 5,000, it freezes network weights to prevent noise overfitting. If variance spikes, it unlocks trust-region plasticity.

### Required Dataset Tracks
Drop the following files into this working folder before triggering the scripts:
* **`Dataset_Train_2018_2019_wdw8.xlsx` & `Dataset_Val_2018_2019_wdw8.xlsx`**: Used to initialize standard scaling layers calibrators.
* **`Dataset_2020_2023_Escalier_Empirique_Am_6h_V3_wdw8.xlsx`**: The target crisis test track generated by the price ladder module.
